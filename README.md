# Verificador de Número Par ou Ímpar

Aplicação em Python que verifica se um número informado pelo usuário é par ou ímpar.

## Funcionalidades
- Entrada de número
- Verificação utilizando operador lógico
- Exibição do resultado

## Tecnologias
- Python

## Objetivo
Praticar estruturas condicionais e operadores matemáticos.
